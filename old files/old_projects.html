First Music Catalog

I made this before I touched javascript. The best way I could come up with was to plant a pdf in an iframe, then all I have to do is ctrl + f to search for a title or artist. This is still my functioning catalog. I have since learned how to use JavaScript to dynamically populate an HTML table with my data. Now I just need the time make it happen.

This was made with HTML and CSS
